# 水獭智聊服务助手小程序

这是水獭智聊的微信小程序服务助手，用于提供帮助内容查询、账号问题自助排查和反馈信息整理功能。

## 版本状态

- `1.1.1`：已提交微信审核，保持当前审核流程，不撤回。
- `1.2.0`：本地品牌视觉统一稿，采用与官网一致的紫粉色系；待 `1.1.1` 审核通过并发布后再上传审核。

## 导入方式

1. 打开微信开发者工具。
2. 选择「导入项目」。
3. 项目目录选择：`shuita-miniprogram`。
4. AppID 使用当前小程序账号：`wx284263ff2fb81977`。

## 当前页面

- 首页：帮助查询、自助排查和反馈整理入口。
- 帮助中心：支持按关键词和类别查询常见问题。
- 自助排查：按问题类型生成后续处理建议。
- 反馈信息整理：在本机生成结构化反馈摘要并连接微信客服。

## 上线前检查

- 小程序后台服务类目与实际服务内容保持一致。
- 小程序备案完成。
- 当前版本不依赖接口，填写内容不会自动上传。
