这是水獭智聊的微信小程序最小可提交版本，用于提供官方下载、安装教程和客服反馈入口。

## 导入方式

1. 打开微信开发者工具。
2. 选择「导入项目」。
3. 项目目录选择：`shuita-miniprogram`。
4. AppID 使用当前小程序账号：`wx284263ff2fb81977`。
